BerkleyDB


Общее описание


Sleepycat Software позиционирует BerkleyDB как встраиваемую библиотеку базы данных с открытым кодом. Данная база данных не является ни реляционной ни объектно-ориентированной. Библиотека имеет размер около 300 килобайт для большинства поддерживаемых архитектур, обеспечивая при этом управление базами данных общей емкостью до 256 тетрабайт. Высокая производительность обеспечивается двумя основными особенностями библиотеки:

  • библиотека встроена непосредственно в приложение, для доступа к базе данных не требуется каких-либо межпроцессорных взаимодействий.
  • программный интерфейс к данным - вызовы функций библиотеки, отсутствует высокоуровневый язык для доступа к данным.
Библиотека базы данных написана на С и поддерживает простое API, основанное на вызовах функций, для большого числа языков программирования, включая C, C++, Java, Perl, Tcl, Python и PHP. Использование библиотеки подразумевает встраивание ее в программу на этапе компиляции (для скриптовых языков вызовы функций библиоткеи из модулей).
